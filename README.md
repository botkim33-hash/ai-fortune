# AI 算命平台 (Fortune AI)

## 项目简介
一个现代化的 AI 算命平台，支持生辰八字排盘、AI 命理解读、双人合盘配对。

## 技术栈
- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui
- **Animation**: Framer Motion
- **AI**: OpenAI API / Claude API
- **Deployment**: Vercel

## 核心功能
1. 🎯 **八字排盘** - 根据出生时间生成八字命盘
2. 🤖 **AI 解读** - 智能分析命理、运势、性格
3. 💑 **双人合盘** - 分析两人配对、姻缘、合作
4. 🎨 **精美设计** - 国风现代融合，设计师级别 UI

## 设计风格
- 国风现代主义
- 深色主题 + 金色点缀
- 流畅动效
- 响应式设计

## 启动开发
```bash
npm install
npm run dev
```

## 部署
```bash
vercel --prod
```
